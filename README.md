# Ping Pong
This Project will display all integers before the users input number as either ping, pong, pingpong, or an integer that isnt divisible by 3,5, or both.
## Tools
I used:
Javascript
Html
CSS
jQuery
## How to Use
Enter a number in the text box and play ping pong.
